# softbody_gift

Multi-physics: 4 stacked tetrahedral soft pyramids wrapped by 2 cloth
strap loops. The straps hold the stack together as it falls.

Demonstrates procedural mesh construction at prep time: `cloth_loop_around_box`
generates the cloth geometry in Python and the vertices are baked into the
recipe JSON.

## How to run

```powershell
cd tests\test_examples\softbody_gift
.\run.ps1
```
